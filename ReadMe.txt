Dit is de ReadMe file voor LeerGo.

LeerGo is een website gemaakt voor de nederlandse GoBond.
Doel: Gebruiker kunnen in 15 min door middel van instructie films en kleine go problemen de regels van het Go spel leren.

De regels zijn onderdverdeeld in verschillende hoofdstukken. Per hoofdstuk is er een instructie film. 
Met daarna een set aan Go problemen.

Technieken

-ccs
-Javascript
-Knockout.js
-Raphael
